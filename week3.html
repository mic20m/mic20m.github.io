Programming

Arduino ide- most user-friendly, many tools, quick, easy
C - intermediate difficult, need to have more hardware knowledge, more efficient in code handling and storing, more freedom to program.
Assembly lowest level of programing, need to have through understanding of the hardware also, more difficult, highly efficient in code handling and occupies least storage, and allows most freedom to program. Basics about programming
